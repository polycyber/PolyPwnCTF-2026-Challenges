# The Last Stronghold

**`Auteur`** [Jean Decian](https://github.com/jeandecian/)

## Description (français)

L'humanité est au bord de l'extinction. Nos principaux scientifiques ont confirmé que les derniers ingrédients biologiques critiques nécessaires au remède sont manquants.
L'analyse des derniers renseignements suggère que ces composants ont été enregistrés pour la dernière fois sur un site sécurisé, et l'un des conteneurs de stockage visibles sur l'image que nous avons récupérée contient ce dont nous avons besoin.

Localisez le site.

Le format du flag est "POLYCYBER{...}" où le site est en lettre minuscule. Dans le cas échéant, remplacer les espaces par des traits d'union. Utilisez le nom du site dans sa langue d'origine.

## Description (english)

Humanity stands on the precipice of extinction. Our lead scientists have confirmed that the final, critical biological ingredients required for the panacea are missing.
Analysis of the latest intel suggests these components were last logged at a secure site, and one of the storage containers shown in our recovered image contains what we need.

Locate the facility.

The flag format is 'POLYCYBER{...}' where the site name is in lowercase letters. If applicable, replace spaces with hyphens. Use the name of the facility in its original language.
