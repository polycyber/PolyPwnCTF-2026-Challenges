## Description (français)

En vous baladant dans le laboratoire, vous tombez sur un ordinateur étrange,
avec une architecture de CPU que vous n'avez jamais vue auparavant. Avec un
peu de recherche, vous trouvez sur Internet un simulateur pour ce CPU :
[Lient GitHub](https://github.com/RomainL972/CodeMachine.git). Le code source
qui nous intéresse est sur la branche `polypwn`.

À côté de l'ordinateur se trouve une disquette qui semble contenir un programme
compilé. Pouvez-vous modifier le comportement du simulateur pour exécuter
ce programme et obtenir le flag ?

## Description (english)

While wandering in the lab, you stumble upon a strange computer, with a CPU
architecture you've never seen before. With a bit of research, you find on the
Internet a simulator for this CPU: [GitHub Link](https://github.com/RomainL972/CodeMachine.git).
The source code we care about is on the `polypwn` branch.

Next to the computer, there's a floppy disk that seems to contain a compiled
program. Can you modify the simulator's behavior to run this program and get the flag?
