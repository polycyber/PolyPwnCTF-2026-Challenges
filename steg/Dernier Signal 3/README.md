# Dernier Signal - Partie 3

**`Auteur`** [Léonard Renoud](https://github.com/Endor508)

## Description (français)

Bon, nous avons le nombre 29556, mais il doit forcément il y avoir plus ! Nos équipes d'exploration ont découvert qu'une des cellules pour les créatures du laboratoire a été forcée de l'intérieur. La cause de l'abandon du laboratoire est forcément expliquée dans la dernière communication. Creuser dans ce fichier audio et découvrer ce qui c'est vraiment passé dans le laboratoire !

Le format du flag est `polycyber{nom de l'entité}`

## Description (english)

Well, we have the number 29556, but there must be more! Our exploration teams discovered that one of the cells for the creatures in the laboratory was forced from the inside. The cause of the abandonment of the laboratory is surely explained in the last communication. Dig into this audio file and discover what really happened in the laboratory!

The flag format is `polycyber{entity name}`
