# Dernier Signal 2

## Write-up FR

Nous pouvons utiliser des logiciels comme Audacity pour voir le spectrogramme du fichier tr@nm!ss!0n_��.wav.

Le spectrogramme montre un code binaire ressemblant à "..--- ----. ..... ..... -...."

Le code morse peut se traduire en utilisant un outil comme CyberChef et donne "29556" qui est le numéro à mettre dans polycyber{...} pour obtenir le flag.

## Write-up EN

We can use software like Audacity to view the spectrogram of the file tr@nm!ss!0n_��.wav.

The spectrogram shows a Morse code pattern resembling "..--- ----. ..... ..... -...."

The Morse code can be translated using a tool like CyberChef and gives "29556", which is the number to put in polycyber{...} to obtain the flag.

## Flag

`polycyber{29556}`
