# Dernier Signal - Partie 1

**`Auteur`** [Léonard Renoud](https://github.com/Endor508)

## Description (français)

Nous venons d'entrer dans le laboratoire abandonné et nos équipes ont découvert un message corrompue. Toutefois, nous n'arrivons pas à comprendre ce qui c'est passé dans le laboratoire. Trouvez la raison de l'abandon du laboratoire.

Le format du flag est `polycyber_...`

## Description (english)

The last communication from the laboratory was intercepted. However, we cannot understand what happened in the laboratory. Find out the reason for the abandonment of the laboratory.

The flag format is `polycyber_...`
