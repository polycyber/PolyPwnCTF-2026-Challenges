## Description (français)

Un script malveillant a été détecté sur la machine. Depuis quel domaine le ficher bat malicieux télécharge-t-il le script?

## Description (english)

Dr. Moreau received a call from "IT support". Since then, his workstation seemed to have been compromised. Uncover what lies beneath the attack.

A malicious script have been detected on the machine. From which domain does the malicious .bat file downloads a script?

Flag format: POLYCYBER{domain.name}
Note: If the URL used is http://poly.pwn/ctf2026/, the flag would be: POLYCYBER{poly.pwn}
