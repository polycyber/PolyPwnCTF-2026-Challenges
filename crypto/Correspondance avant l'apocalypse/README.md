# Correspondances avant l'apocalypse

**`Auteur.e`** Dora

## Description FR

Dans le fond d'un tiroir du laboratoire, j'ai trouvé des lettres manuscrites, qui semblaient être un échange entre plusieurs scientifiques ... ils évoquaient une probable fin du monde ... la découverte d'une terrible puissance ... 
Tout cela est inquiétant, mais la dernière lettre est illisible, elle doit surement contenir une information essentielle.


## Description EN

At the bottom of a laboratory drawer, I found handwritten letters that appeared to be an exchange between several scientists… they spoke of a possible end of the world… the discovery of a terrible power…
All of this is deeply unsettling, but the last letter is unreadable. It must certainly contain a crucial piece of information.
