# Nom du défi
DexXOR's Lab

**`Auteur·e`** [Hib00boo](https://github.com/Hib00boo)

## Description (français)

Vous avez réussi à rentrer dans votre laboratoire, mais les systèmes de défense se sont déclenchés. Heureusement, vous gardez toujours le mot de passe d'arrêt dans votre sarrau. Malheureusement, vous l'avez crypté à l'aide de mathématiques de pointe que seul un jeune génie pourrait comprendre. Ainsi, votre sœur ne peut pas entrer dans votre laboratoire sans votre permission. Utilisez vos clés de sécurité pour découvrir le mot de passe.

## Description (english)

You managed to get back into your lab but the defense systems tripped. Thankfully, you always keep the shutdown password in your lab coat. Unfortunately, you encrypted them using super high tech math that only a boy genius would understand. That way your sister can't get into your lab without your permission. Use your security keys to uncover the password.
