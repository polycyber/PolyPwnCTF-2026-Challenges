# Nom du défi
Dee Dee-coded

## Write-up

The cipher on the picture looks to be using a substitution cipher algorithm, since letters repeat and both uppercase and lowercase letters are included. The cipher was written using a simple vigenere cipher algorithm (can be detected by pasting ciphertext on dcode.com). Solving it requires either using cyberchef with the key "DEEDEE" or writing a decoder and using said key.

Le chiffrement sur l'image semble utiliser un algorithme de chiffrement par substitution, car les lettres se répètent et comprennent à la fois des majuscules et des minuscules. Le chiffrement a été écrit à l'aide d'un algorithme de chiffrement Vigenère simple (qui peut être détecté en collant le texte chiffré sur dcode.com). Pour le déchiffrer, il faut soit utiliser Cyberchef avec la clé « DEEDEE », soit écrire un décodeur et utiliser la clé DEEDEE.

Cipher : 'Eiadvi, isv wlmv mw rrpb xlh fijmrqmrj... TSOCGBFIU{QeQHEuObGIigIIIsVhZIu}'
Plaintext : 'Beware, for this is only the beginning... POLYCYBER{MaNDArKxDEedEEFoReVEr}'
